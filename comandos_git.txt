GIT

git config --global user.name "nombre de usuario"
git config --global user.email "correo del usuario"

git init (agregar git a nuestra carpeta)
git status (rastrear cambios)
git add -A (Incluir/preparar archivos modificados para el guardado)
git commit -m "descripcion del cambio" (Guardar los cambios)
git checkout -- . (recuperar archivos del ultimo commit)
git push origin master (subir los cambios a un hosting)
git clone URL (clonar repo desde el hosting)



mkdir "nombre" (crear nueva carpeta)
touch "nombre.extension" (crear un archivo nuevo)
